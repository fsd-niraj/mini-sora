# Mini Sora

### This is a replication of text-to-video model (Sora) for educational purposes.

### Fork and run

```
(Fork)[https://github.com/fsd-niraj/mini-sora/fork] this repo.
cd mini-sora

pip3 install requirements.txt

streamlit run app.py
```

This will open up a streamlit instance in a browser and provide a simple chat like UI for prompts and videos.

This project Still in progress.
